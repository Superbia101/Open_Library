Крестики-Нолики - игра крестики-нолики, на соснове ООП

abstract_class - пример и образец реализации абстрактного класса

karma - забавный пример создания и использования своих классов исключений

magic - забавная концепуия классов-стихий

matrix - небольшой класс для обработки матриц

my_dict - небольшой пример перегрузки стандартных методов

primes - класс-итератор, который принимает максимальное число N и выдаёт все простые числа от 1 до N. 

stack - пример реализации наследования и создания своей структуры типа данных - стек

linked_list - своя реализация односвязного списка
