Небольшая подборка созданных мной интересных функций Python 


add_symmetrical_seq - Определяет минимальное количество и каких чисел надо приписать в конец последовательности, чтобы она стала симметричной.

archives_the_string - Принимает и кодирует строку, заменяя символы их количеством.

binary_search - поиск числа в отрезке, с указаниями через клавиатуру и алгоритм бинарного поиска

bubble_sorting - cортировка методом пузырька в порядке возрастания

caesar_cipher - Шифровка переданной строки кодом Цезаря, с заданным сдвигом.

check_exponent_degree - 2 функции: проверяют изменится ли показатель степени e при сложении двух чисел

check_ip, check_file - 2 функции: 1-я проверяет строку на бытие ip-адресом; 2-я проверяет список со строкой на соответствие условиям.

check_palindrome_substitution - Определяет, существует ли у переданной строки такая перестановка, при которой она станет палиндромом.

classify_triangle - определяет может ли существовать треугольник с такими сторонами, если да - считает площадь по формуле Герона

comparison_real_numbers - принимает три числа и затем сравнивает сумму первых двух чисел с третьим 
с определённой степенью точности: до 15-го знака после точки

conversion_number - Переводит число в необходимую систему счисления до 36-й

count_letters_and_nums - Поиск и подсчёт буквы и цифры в тексте

count_prime_numbers, prime_num, prime_num_list - 3 функции: 1-я Считает количество простых чисел в заданной последовательности.
2-я Проверка переданного числа на принадлежность к простым. 3-я Создаёт список из простых чисел до переданного.

deep_copy_and_replace, creating_website - 2 функции: 1-я Производит глубокое копирование заданного словаря и производит 
замену определённых строк на предоставленную. 2-я Создаёт и выводит (намеренно введённый mutable list) список созданных макетов.

dict_freq, dict_freq_invert, dict_print - 3 функции: Создаёт из переданной строки словарь частот символов. 
Создаёт из переданного словаря инвертированный по отношению ключ - значение.
Выводит на экран элементы переданного словаря.

disclosure_list - Раскрывает многомерный список любой вложенности.

division_5_remainder_3 - несколько разных вариантов одной идеи: создание и вывод списка 
чисел - имеющих общие делитель с опред. остатком	

fib - вывод числа под заданным номером из последовательности Фибоначчи 

files_size_in_dir - Получает на вход путь до каталога и выводит общее количество файлов и подкаталогов в нём.
Также выведите на экран размер каталога в килобайтах.

find_key - Поиск во вложенном словаре записи по ключу.

frequency_analysis, zip_output - 2 функции: 1-я выполняет частотный анализ и выводит отсортированный результат в файл analysis.txt, регистр символов учитывается.
2-я распаковывает в текущий каталог архив.

game - группа функций изобржают игру	

gen_files_path - Функция-генератор которая рекурсивно проходит по всем каталогам (включая вложенные папки и подпапки) указанной
    директории (по умолчанию — корневой диск), находит указанный пользователем каталог и генерирует пути всех
    встреченных файлов.

str_count_in_dir - Функция-генератор, берёт все файлы .py в директории и вычисляет количество строк в каждом файле,
    игнорируя пустые строки и строки комментариев.

graphics_pit - получает на вход число N и выводит на экран числа в виде ямы

graphics_pyramid - выводит на экран пирамидку, заполненную символами хештега(#)

graphics_pyramid_num - выводит на экран заполненную нечётными числами пирамиду

intersection_lists, difference_lists - 2 функции: Возвращает список из элементов которые присутствуют во всех переданных списках. 
Возвращает список из элементов первого списка, которых нет во всех переданных остальных списках.

line_palindrome_check - проверяет строку является ли она палиндромом

list_comparison - сравнение на предмет равенства два числовых списка

list_element_insert - между каждой парой элементов этого списка вставляется новый элемент, 
равный сумме значений соседних элементов

list_sort_special - список заполняется случайными числами. и преобразовывается в перемежающуюся последовательность

matrix_random_letters - создание и вывод матрицы, случайным образом заполненной буквами от A до Z

matrix_redact - cоздаёт и выводит вложеный список n на m, у котор. удаляется опред строка и столбец

matrix_snake - cоздаёт и выводит матрицу заполненную числами по спирали

merge_sorted_lists - Объединяет два отсортированных списка чисел в один отсортированный без дубликатов

my_sum - Складывает числовые значения переданных аргументов (словари, списки, кортежи).

node_euclid_algorithm - поиск НОД алгоритмом Евклида

output_content_number - определяет сколько в целом числе различных цифр и выводит на экран

output_degrees_of_two - выводит необходимое кол-во степеней 2

output_factorial - факториал заданного числа

output_nums - Выводит на экран все числа от 1 до переданного. Рекурсивно!

output_param_func - max и min значение переданной функции в целочисленных точках диапазона

output_running_nums - Сдвигает элементы списка/строки вправо на заданное кол-во позиций

quicksort_hoare - Алгоритм быстрой сортировки (сортировка Хоара), опорный элемент крайний правы.

replacement_for - Эмулирует работу цикла for с помощью цикла while и проходит по всем элементам переданного итерируемого объекта.

retiring_list - выбывание элементов из списка по мтоду "считалочки"

search_key_indict - Находит заданный пользователем ключ в словаре и выдаёт значение этого ключа, 
возможна настройка глубины поиска. (Словарь должен быть в области видимости по идентификатором site)

second_max - возвращает первый и второй максимум и, их индексы из переданного списка

special_numbers - создание и вывод интервала чисел - имеющих общие делитель с остатком, без остатка и 
сумма цифр которых меньше заданного

sum_multiplic_list -  получает сумму попарных произведений элементов списков. если в одном из списков элементов меньше, 
то недостающие элементы получаются путем циклического повторения содержимого списка.

sum_odd - возвращает сумму нечётных чисел от нуля до заданного

timer - считает минимальное время выполнения переданой функции

сhange_working_directory - Функция контекст-менеджер, временно меняет текущую рабочую директорию на переданную.
